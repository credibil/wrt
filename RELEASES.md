## 0.17.0

Released 2025-11-18

### Changed

- Aligned env vars with documentation
- Updated dependencies to latest versions
- Renamed `realtime-messaging` to `realtime` and added `keyvalue` store support for a 
  'kitchen-sink' build.

**Full Changelog**: https://github.com/credibil/wrt/compare/v0.4.0...v0.17.0
